Dear users,

Here is a wonderful application which count the number of cars per second displayed on a video.

In order to count, you should load a video, for that click on File -> Open Video or use CTRL+O.

Then you just have to click on play,then the video and the processing will start. You can pause the video at any moment with the pause button or reset the video with the stop button.

To have the best result you can adjust some parameters displayed on the right panel of the windows. Each of them have a tool tip to help you understand what it change for our algorithms.

And you can visualize the different algorithm running thanks to the combo box.

There is 3 algorithm:

Background: display the background computed.

Motion: display the moving parts of the video.

Contours: display the contours of the result of the motion algorithm.


You can save/load a project using File -> Save Project/Load Project.

And you can quit the application using CTRL+Q or File -> Quit

Thank you and have fun with this App'
